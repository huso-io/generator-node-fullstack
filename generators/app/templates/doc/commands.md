# 指令

## 监测修改并自动跨设备同步刷新浏览器

```sh
$ yarn start
```

## 构建 & 优化

```sh
$ yarn build
```

完全重新构建整个工程以达到可发布状态，这当中包括诸如代码检测、图片优化、样式表等静态资源的优化及混合等。

## 完全构建 & 优化整个网站并启动服务器预览

```sh
$ yarn preview
```

## 代码检测

```sh
$ yarn lint
```

## 执行所有测试用例

```sh
$ yarn test
```

## 生成更新日志

```sh
$ yarn changelog
```

## 发布

```sh
$ yarn release
```

## 上线

```sh
$ yarn online
```
